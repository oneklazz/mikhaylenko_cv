## описание работы программы
программа делает непрерывные скриншоты области экрана, где расположена игра T-Rex Dinosaur Game
### полученное изображение проходит несколько этапов обработки:
1. захват изображения с экрана
2. перевод изображения в оттенки серого
3. бинаризация изображения
3. анализ определённых зон
4. принятие решения
5. выполнение действия
### если программа обнаруживает препятствие:
выполняется прыжок
либо пригибание
## методы компьютерного зрения
в проекте используются базовые методы компьютерного зрения
### перевод изображения в grayscale
изображение переводится в чёрно-белый формат для уменьшения количества данных и ускорения обработки.
### бинаризация изображения
используется функция threshold, которая разделяет изображение на:
* белые объекты
* чёрный фон
### анализ области интереса
бот проверяет две специальные области:
* ground_zone — препятствия на земле
* air_zone — препятствия в воздухе
если количество белых пикселей превышает заданный порог, программа считает что обнаружено препятствие
## используемые библиотеки
1. opencv (cv2) - обработка изображения
2. numpy - работа с массивами
3. mss - быстрый захват экрана
4. pyautogui - управление клавиатурой
5. time - задержки
## структура программы
### программа состоит из нескольких этапов:
1. запуск программы
2. ожидание открытия игры
3. захват изображения
4. обработка изображения
5. поиск препятствий
6. управление персонажем
7. отображение результата
## как выполнялся проект
проект выполнялся двумя участниками
### первый участник
* реализовал обработку изображения
* настраивал бинаризацию
* подбирал параметры зон проверки
* занимался компьютерным зрением
### второй участник
* реализовал управление клавиатурой
* настроил захват экрана
* тестировал алгоритм
* оптимизировал скорость работы программы
